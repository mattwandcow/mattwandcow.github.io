Target Audience

Our target audience is people who want to be able to plan far ahead when making plans, and have contingencies so that events will never catch them unawares.

Personas

1. Jackson is planning a camping trip, and he wants to make sure he's going to have clear nights to allow for stargazing while camping.

2. Jennifer likes hiking, but not when it's too hot. She wants to make sure her upcoming hike is going to be comfortable.

Scenerios

1. The day looks nice. Should I carry a jacket anyway?

2. It looks like rain this weekend. When would the weather be clear again?

3. If I put off shovelling snow for another day, would it melt or freeze?

4. What is nearby if an activity is rained out?

5. Is there anyplace nearby to swim?
